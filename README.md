### 🚀 Featured Projects

**[Nexora](https://github.com/mo74x/Nexora)** — Multi-Tenant Event Ingestion & Analytics Platform
A 3-service microservices platform communicating via gRPC and Kafka, with atomic per-tenant rate limiting (Redis Lua token-bucket) and a GraphQL analytics API. Load-tested with k6 at sub-500ms p99 latency under 200 concurrent users.

**Caliber Talent Network** — Event-Driven Recruitment Platform
Async, event-driven resume processing pipeline using Redis + BullMQ, with Stripe webhook integration and JWT-based RBAC for strict recruiter/candidate data isolation.

**PitWall** — Real-Time F1 Telemetry System
High-concurrency microservices system processing real-time race telemetry via RabbitMQ, with a Redis-backed live leaderboard and AWS ECS Fargate auto-scaling deployment.

**LogiTrack** — Event-Driven Logistics Backend
Real-time delivery tracking backend using Kafka for event streaming and MongoDB geo-spatial queries for driver-order matching, with Socket.io live location updates.
